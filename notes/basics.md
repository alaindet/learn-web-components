# Web Components

Web Components are based on three major technologies (four including a discontinued one)

### Custom HTML Elements
- They allow you to register new HTML tags into the browser

### Shadow DOM
- Completely separate DOM node tree into custom HTML elements
- Has scoped CSS

### Templates & Slots
- Define HTML structure hidden inside Custom HTML Elements
- Allow user to fill the slots without knowing the final structure

### HTML Imports (discontinued)
- HTML imports inside HTML file
